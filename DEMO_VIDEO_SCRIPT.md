## SECTION 6: TECH ARCHITECTURE (3:30 - 4:00)

**[Screen: Show a simple diagram or just narrate]**

🎙️ NARRATION:
"Under the hood, DocGen AI is built with Next.js 16, TypeScript, and Tailwind CSS.
When you click Generate, it fires TWO API calls in parallel —
one for documentation, one for personality analysis —
using Promise.allSettled so if one fails, the other still works.
Both calls go to the AI through a secure backend — no API keys exposed to the frontend.
The result? Fast, resilient, and elegant."

## RECORDING TIPS:
# 1. Use Chrome in full screen (hide bookmarks bar with Ctrl+Shift+B)
# 2. Set zoom to 100% or 125% for best readability
# 3. Use a dark theme for the recording — it looks more professional
# 4. Practice the flow 2-3 times before recording
# 5. Keep the recording under 4 minutes — judges have many submissions
# 6. Speak clearly and with energy — show you're excited about what you built
# 7. Upload to YouTube (Unlisted) or Google Drive (anyone with link can view)
